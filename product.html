Product Overview

Enterprise diagnostic engine for marketing KPI analysis

How It Works

AFX analyzes your marketing KPI data to identify performance patterns and root causes:

  1. Symptom Detection: Identifies abnormal KPI movements
  2. Pattern Matching: Matches symptoms to known performance patterns
  3. Root Cause Inference: Determines underlying mechanisms
  4. Safe Output Generation: Produces actionable recommendations

Key Features

  • 1000+ canonical KPI metrics supported
  • 30+ diagnostic patterns
  • Real-time analysis and recommendations
  • Enterprise-grade security and compliance
  • PDPA, GDPR, and PCI compliance

Supported KPIs

AFX supports diagnostic analysis across multiple KPI categories:

  • Email metrics (CTR, open rate, bounce rate)
  • Social media metrics (reach, engagement, follower growth)
  • Conversion metrics (funnel conversion, cart abandonment)
  • Revenue metrics (ROAS, CPA, revenue per session)
  • Retention metrics (churn rate, lifetime value)
