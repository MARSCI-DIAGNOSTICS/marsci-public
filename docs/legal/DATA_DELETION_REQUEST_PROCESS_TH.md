# กระบวนการคำขอลบข้อมูล

เอกสารนี้เป็นคำแปลภาษาไทยของฉบับภาษาอังกฤษ
หากมีความไม่สอดคล้องกัน ให้ยึดฉบับภาษาอังกฤษเป็นหลัก

อัปเดตล่าสุด: 6 มีนาคม 2026

หน้านี้เป็นร่างกระบวนการสำหรับ MarSci Diagnostics โดยอธิบายว่าผู้ใช้อาจยื่นคำขอเข้าถึง แก้ไข ลบ หรือขอรับข้อมูลส่วนบุคคลได้อย่างไร ทั้งนี้ขึ้นอยู่กับกฎหมายที่ใช้บังคับ ความเป็นไปได้ทางเทคนิค และข้อจำกัดด้านการปฏิบัติการ

## 1. ประเภทของคำขอ

ผู้ใช้อาจติดต่อ MarSci Diagnostics เพื่อ:

- ขอเข้าถึงข้อมูลส่วนบุคคล
- ขอแก้ไขข้อมูลที่ไม่ถูกต้อง
- ขอให้ลบข้อมูล
- ขอรับข้อมูลในรูปแบบที่โอนย้ายได้หรือ export ได้

## 2. วิธีการยื่นคำขอ

สามารถส่งคำขอมาที่:

- support@marsci-diagnostics.com

โปรดให้ข้อมูลเพียงพอเพื่อให้ MarSci Diagnostics สามารถระบุบัญชีและทำความเข้าใจคำขอได้

## 3. กระบวนการยืนยันตัวตน

ก่อนดำเนินการตามคำขอ MarSci Diagnostics อาจตรวจสอบตัวตนและอำนาจของผู้ยื่นคำขอ และอาจขอข้อมูลเพิ่มเติมเมื่อจำเป็นอย่างสมเหตุสมผลเพื่อป้องกันการเปิดเผย การลบ หรือการแก้ไขข้อมูลโดยไม่ได้รับอนุญาต

## 4. ระยะเวลาการตอบกลับ

MarSci Diagnostics อาจตอบกลับภายในระยะเวลาที่สมเหตุสมผล โดยคำนึงถึงประเภทของคำขอ ข้อกำหนดตามกฎหมาย ความซับซ้อนของบัญชี และข้อมูลที่มีอยู่ ระยะเวลาที่แน่นอนยังไม่ถูกกำหนดอย่างสมบูรณ์ในร่างฉบับนี้ และอาจขึ้นอยู่กับกฎหมายที่ใช้บังคับ

## 5. ข้อจำกัดที่อาจเกิดขึ้น

ข้อมูลบางประเภทอาจไม่สามารถลบ ส่งออก หรือแก้ไขได้ทันที โดยเฉพาะเมื่อจำเป็นต้องเก็บไว้เพื่อ:

- ภาระหน้าที่ตามกฎหมาย
- ข้อมูลการเรียกเก็บเงินและบัญชี
- การป้องกันการทุจริต
- การสอบสวนด้านความมั่นคงปลอดภัย
- ข้อกำหนดด้านการตรวจสอบ
- การจัดการข้อพิพาท
- ข้อจำกัดด้าน backup ทางเทคนิคหรือความสมบูรณ์ของระบบ

## 6. ผลกระทบต่อการใช้งานบริการ

คำขอให้ลบหรือจำกัดการประมวลผลข้อมูลอาจส่งผลต่อการเข้าถึงบางส่วนหรือทั้งหมดของบริการ หากข้อมูลดังกล่าวเป็นส่วนจำเป็นต่อการดำเนินงานของบัญชีหรือการให้บริการ

## 7. หมายเหตุเพิ่มเติม

กระบวนการนี้อาจมีการปรับปรุงเมื่อ MarSci Diagnostics พัฒนาเครื่องมือสำหรับการจัดการบัญชี การ export ข้อมูล หรือ workflow การลบข้อมูลเพิ่มเติม
