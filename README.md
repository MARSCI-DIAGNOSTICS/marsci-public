# MarSci Diagnostics Public Repository

This repository contains public-safe documentation, legal pages, brand materials, and interface-level product information for MarSci Diagnostics.

MarSci Diagnostics is a KPI Diagnostic System for marketing performance analysis. It is not a campaign optimizer, advertising automation system, or media-buying platform.

Included here:
- public product documentation
- legal and policy pages
- public architecture overview
- brand/color materials

Not included here:
- internal detection logic
- proprietary diagnostic methodology
- scoring systems
- connector credentials
- runtime engine code
- internal datasets or mappings
